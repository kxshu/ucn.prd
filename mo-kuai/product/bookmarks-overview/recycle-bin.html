页面-2.2.3 recycle bin

定义

recycle bin 界面为被移除的产品总集，主要分为侧边栏，搜索条件和搜索结果三个板块

1. 侧边栏的按钮及对应规则

板块 操作 操作结果
Quick Linkling order overview 跳转到8.1 order overview
logistic plan overview 跳转到10.1 logistic plan overview
warehouse overview 跳转到9.1 warehouse overview
inquiry overview 跳转到4.1 inquiry overview
add links 跳出快捷链接选择弹窗，弹窗样式如下，字段详见字段说明表
pages log 新窗口打开log页面，log指当前页面的操作日志，页面样式参考原型log页面
Action create inquiry 跳转到inquiry的创建界面4.3-create inquiry
create order 跳转到order创建界面 8.2-create order
download the overview 下载当前回收站内的所有产品
快捷链接的弹窗样式

2. 搜索条件字段及操作

① 搜索条件字段详见字段说明表

② 操作

按钮 操作结果
advanced 展开高级搜索，展开后，按钮变为hide the advanced
search 根据所选的搜索条件进行搜索
clear 清除搜索条件

3. 搜索结果及对应操作

① 搜索结果字段详见字段说明表 ② 页码设置：默认每页显示25条数据，用户可自己选择每页显示的数量，可选项目为 10条/页，20条/页，50条/页，100条/页，200条/页，500条/页，1000条/页，2000条/页 ③ 固定列：首列的check box 和 尾列的 Action

按钮 操作结果
☴数据筛选 针对当前字段中数据的筛选及排序：筛选为当前页筛选，排序为所有页面排序。
☸字段设置 针对当前表格的字段显示设置，根据用户保存设置结果，即某用户设置了部分字段隐藏后，关闭界面，下次再次进入，显示为设置后的字段结果
Recover 将选中的产品还原到之前的地方（如果从bookmark来的就回到bookmark去），未选产品此按钮为灰掉状态，不可点击；点击了该按钮后，页面不跳转，而是直接将已经还原的产品从列表中移除，并给出提示“successfully recovery”
列表中的action detail：点击进入查看该产品的详情页
